Authentication

Learn how to authenticate your API requests securely

API Keys

Nexus API uses API keys to authenticate requests. You can view and manage your API keys in the Nexus Dashboard.

# Include your API key in the Authorization header curl https://api.nexusapi.dev/v1/me \ -H "Authorization: Bearer YOUR_API_KEY"

Key Types

TypePrefixUse Case
Secret Keynxk_live_Server-side requests
Publishable Keynxk_pub_Client-side (limited access)
Test Keynxk_test_Development and testing
⚠️
Keep your keys secure!
Never expose secret keys in client-side code or public repositories. Use environment variables instead.

SDK Authentication

# TypeScript / JavaScript import { NexusClient } from '@nexusapi/sdk'; const nexus = new NexusClient({ apiKey: process.env.NEXUS_API_KEY });
# Python from nexusapi import NexusClient import os client = NexusClient( api_key=os.environ.get('NEXUS_API_KEY') )

Environment Variables

Create a .env file in your project root:

# .env file NEXUS_API_KEY=nxk_live_xxxxxxxxxxxx
💡 Tip: Make sure to add .env to your .gitignore file to prevent accidentally committing secrets.

Key Rotation

To rotate an API key:

  1. Go to Settings → API Keys in your dashboard
  2. Click the key you want to rotate
  3. Click Rotate Key
  4. Update your environment variables with the new key
  5. Verify everything works
  6. Delete the old key
