Users API

Manage user accounts, profiles, and permissions

Base URL

https://api.nexusapi.dev/v1/users

Endpoints

GET /me

Get the authenticated user's profile

GET /users/:id

Get a user by their ID

GET /users

List all users in your organization

POST /users

Create a new user

PATCH /users/:id

Update an existing user

DELETE /users/:id

Delete a user

Get Current User

# Request curl https://api.nexusapi.dev/v1/me \ -H "Authorization: Bearer YOUR_API_KEY" # Response { "success": true, "data": { "id": "usr_abc123", "email": "developer@example.com", "name": "Alex Developer", "role": "admin", "plan": "pro", "created_at": "2024-01-01T00:00:00Z" } }

Create User

# Request curl -X POST https://api.nexusapi.dev/v1/users \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "email": "newuser@company.com", "name": "New User", "role": "member" }'

User Object

id string Unique user identifier
email string User's email address
name string User's display name
role string User role: owner, admin, or member
plan string Subscription plan: free, pro, or enterprise
created_at string ISO 8601 timestamp
