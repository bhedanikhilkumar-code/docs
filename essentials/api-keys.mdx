API keys authenticate your requests to Nexus API.

## Creating an API Key

1. Log in to [Nexus Dashboard](https://app.nexus.dev)
2. Go to **Settings** → **API Keys**
3. Click **Create New Key**
4. Copy and store the key securely

<Warning>
  API keys are shown only once. Store them securely immediately.
</Warning>

## Key Types

```bash
# Live key
nx_live_1a2b3c4d5e6f7g8h9i0j

# Sandbox key
nx_test_1a2b3c4d5e6f7g8h9i0j

# Restricted key
nx_restricted_1a2b3c4d5e6f7g8h9i0j
```

## Usage

```python
import os
from nexus import Client

client = Client(api_key=os.getenv("NEXUS_API_KEY"))
```

## Security Checklist

- ✓ Stored in environment variables
- ✓ Not in frontend code
- ✓ Rotated periodically
- ✓ Restricted permissions
