---
title: Metrics
description: Query detailed analytics metrics
---

Retrieve detailed metrics with time-series data.

## Endpoint

```
GET /v1/analytics/metrics
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `metric` | string | Yes | Metric type: `requests`, `latency`, `errors`, `users` |
| `period` | string | No | Time period: `24h`, `7d`, `30d`, `90d` |
| `interval` | string | No | Granularity: `hour`, `day`, `week` |
| `filter` | object | No | Filter conditions |

## Request

```bash
curl "https://api.nexus.dev/v1/analytics/metrics?metric=requests&period=7d&interval=day" \
  -H "Authorization: Bearer nx_live_xxxxxxxxxxxx"
```

## Python SDK

```python
# Get request metrics
requests = client.analytics.metrics(
    metric="requests",
    period="30d",
    interval="day"
)

for point in requests.data:
    print(f"{point.date}: {point.value} requests")

# Get latency metrics
latency = client.analytics.metrics(
    metric="latency",
    period="7d"
)
print(f"Average latency: {latency.summary.average}ms")

# Filter by endpoint
errors = client.analytics.metrics(
    metric="errors",
    period="7d",
    filter={"endpoint": "/v1/users"}
)
```
