---
title: List Users
description: Retrieve a paginated list of all users
---

Retrieve a list of all users in your account. Results are sorted by creation date, newest first.

## Endpoint

```
GET /v1/users
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `limit` | integer | No | Number of results (1-100, default: 20) |
| `cursor` | string | No | Pagination cursor |
| `status` | string | No | Filter by status: `active`, `inactive`, `suspended` |
| `role` | string | No | Filter by role: `owner`, `admin`, `member`, `viewer` |
| `sort` | string | No | Sort field: `created_at`, `updated_at`, `email`, `name` |
| `order` | string | No | Sort order: `asc`, `desc` |

## Python SDK

```python
from nexus import Client

client = Client(api_key="nx_live_xxxxxxxxxxxx")

# List all users with auto-pagination
for user in client.users.list_all():
    print(f"{user.name} ({user.email})")

# Filter by status
active_users = client.users.list(status="active")

# Sort by name
users = client.users.list(sort="name", order="asc")
```
