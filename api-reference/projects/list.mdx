---
title: List Projects
description: Retrieve all projects in your account
---

Get a paginated list of all projects.

## Endpoint

```
GET /v1/projects
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `limit` | integer | No | Number of results (1-100, default: 20) |
| `cursor` | string | No | Pagination cursor |
| `status` | string | No | Filter by status: `active`, `archived` |
| `visibility` | string | No | Filter by visibility: `public`, `private`, `internal` |

## Request

```bash
curl "https://api.nexus.dev/v1/projects?limit=20" \
  -H "Authorization: Bearer nx_live_xxxxxxxxxxxx"
```
