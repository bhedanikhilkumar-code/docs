This guide will walk you through the essential steps to start using Nexus API.

## Prerequisites

- A Nexus API account ([sign up free](https://app.nexus.dev/signup))
- An API key (see [API Keys](/essentials/api-keys))
- Basic familiarity with REST APIs

## Step 1: Get Your API Key

1. Log in to your [Nexus Dashboard](https://app.nexus.dev)
2. Navigate to **Settings** → **API Keys**
3. Click **Create New Key**
4. Copy your key — you'll only see it once!

<Warning>
  Never expose your API key in client-side code or public repositories. Use environment variables.
</Warning>

## Step 2: Install the SDK

<Tabs>
  <Tab value="python" label="Python">

  </Tab>
  <Tab value="node" label="Node.js">

  </Tab>
</Tabs>

## Step 3: Make Your First Request

Let's verify your API key by fetching your user profile:

<Tabs>
  <Tab value="python" label="Python">
    ```python
    from nexus import Client
    
    client = Client(api_key="nx_live_xxxxxxxxxxxx")
    
    # Get current user
    me = client.users.me()
    print(f"Logged in as: {me.email}")
    ```
  </Tab>
  <Tab value="node" label="Node.js">
    ```javascript
    import { NexusClient } from '@nexus/sdk';
    
    const client = new NexusClient({ apiKey: 'nx_live_xxxxxxxxxxxx' });
    
    // Get current user
    const me = await client.users.me();
    console.log(`Logged in as: ${me.email}`);
    ```
  </Tab>
  <Tab value="curl" label="cURL">
    ```bash
    curl https://api.nexus.dev/v1/users/me \
      -H "Authorization: Bearer nx_live_xxxxxxxxxxxx"
    ```
  </Tab>
</Tabs>

**Expected Response:**
```json
{
  "id": "usr_1a2b3c4d5e",
  "email": "developer@example.com",
  "name": "Developer",
  "created_at": "2024-01-15T10:30:00Z"
}
```

## Step 4: Create Your First Resource

Create a project to organize your work:

```python
project = client.projects.create(
    name="My First Project",
    description="Getting started with Nexus API"
)
print(f"Created project: {project.id}")
```
