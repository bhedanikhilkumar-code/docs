Files API

Upload, download, and manage files with built-in storage and CDN delivery

Base URL

https://api.nexusapi.dev/v1/files

Endpoints

GET /files

List all files

POST /files/upload

Upload a file (multipart/form-data)

GET /files/:id

Get file metadata

DELETE /files/:id

Delete a file

Upload File

curl -X POST https://api.nexusapi.dev/v1/files/upload \ -H "Authorization: Bearer YOUR_API_KEY" \ -F "file=@./document.pdf" \ -F "name=document.pdf" \ -F "project_id=prj_abc123" # Response { "success": true, "data": { "id": "file_xyz789", "name": "document.pdf", "size": 102400, "url": "https://cdn.nexusapi.dev/files/xyz789.pdf" } }

Supported File Types

TypeExtensionsMax Size
Imagesjpg, png, gif, webp, svg10 MB
Documentspdf, doc, docx, txt, md50 MB
Archiveszip, tar, gz, rar100 MB
Videomp4, mov, avi, webm500 MB
Audiomp3, wav, flac, ogg100 MB

CDN Delivery

All uploaded files are automatically served through our global CDN for fast delivery.

# Access your file via CDN https://cdn.nexusapi.dev/files/file_xyz789.pdf # Or generate a signed URL (expires in 1 hour) GET /v1/files/:id/signed-url
