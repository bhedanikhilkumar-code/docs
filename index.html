⚡ Quick Start

Get your API key and make your first request in under 5 minutes.

# Install the SDK npm install @nexusapi/sdk # Make your first request import { NexusClient } from '@nexusapi/sdk'; const nexus = new NexusClient({ apiKey: process.env.NEXUS_API_KEY }); const user = await nexus.users.me();
