Projects API

Create and manage projects, organize resources, and track progress

Base URL

https://api.nexusapi.dev/v1/projects

Endpoints

GET /projects

List all projects in your organization

POST /projects

Create a new project

GET /projects/:id

Get a project by ID

PATCH /projects/:id

Update a project

DELETE /projects/:id

Delete a project

List Projects

# Request curl https://api.nexusapi.dev/v1/projects \ -H "Authorization: Bearer YOUR_API_KEY" # Response { "success": true, "data": [ { "id": "prj_abc123", "name": "My First Project", "description": "A sample project", "status": "active", "created_at": "2024-01-01T00:00:00Z" } ], "total": 1 }

Create Project

# Request curl -X POST https://api.nexusapi.dev/v1/projects \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "name": "New Project", "description": "Project description here" }' # Response { "success": true, "data": { "id": "prj_new123", "name": "New Project", "status": "active" } }

Project Object

id string Unique project identifier
name string Project name
description string Project description
status string active, paused, or archived
created_at string ISO 8601 timestamp
