## Environment Setup

### 1. Environment Variables

Create a `.env` file in your project root:

```bash
# Nexus API Configuration
NEXUS_API_KEY=nx_live_xxxxxxxxxxxx
NEXUS_BASE_URL=https://api.nexus.dev
NEXUS_WEBHOOK_SECRET=whsec_xxxxxxxxxxxx

# Optional: Use sandbox for testing
NEXUS_ENV=sandbox
```

<Note>
  In production, use a secrets manager like AWS Secrets Manager or HashiCorp Vault.
</Note>

### 2. SDK Configuration

<Tabs>
  <Tab value="python" label="Python">
    ```python
    import os
    from nexus import Client
    
    # Initialize with environment variables
    client = Client(
        api_key=os.getenv("NEXUS_API_KEY"),
        timeout=30,
        max_retries=3
    )
    ```
  </Tab>
  <Tab value="node" label="Node.js">
    ```javascript

      apiKey: process.env.NEXUS_API_KEY,
      timeout: 30000,
      maxRetries: 3
    });
    ```
  </Tab>
</Tabs>

## Local Testing

### Sandbox Environment

Use our sandbox for testing without affecting production data:

```bash
# Sandbox API URL
NEXUS_BASE_URL=https://api-sandbox.nexus.dev
```

Sandbox features:
- All endpoints available
- Data is isolated and resettable
- Rate limits are relaxed
- No real charges

### Testing Webhooks Locally

Use [ngrok](https://ngrok.com) to test webhooks locally:

```bash
# Start ngrok tunnel to port 3000
ngrok http 3000

# Update webhook URL in dashboard to:
# https://abc123.ngrok.io/webhooks/nexus
```

## Error Handling

Implement robust error handling:

```python
from nexus.exceptions import (
    NexusAuthError,
    NexusRateLimitError,
    NexusServerError,
    NexusNotFoundError
)

try:
    user = client.users.get("usr_123")
except NexusAuthError:
    print("Invalid API key")
except NexusRateLimitError:
    print("Rate limited - implement backoff")
except NexusNotFoundError:
    print("User not found")
except NexusServerError:
    print("Server error - contact support")
```

## Performance Optimization

### Caching

Implement caching for frequently accessed data:

```python
import cachelib

cache = cachelib.Cache(ttl=300)  # 5 minute TTL

def get_user_cached(user_id):
    cache_key = f"user:{user_id}"
    if cached := cache.get(cache_key):
        return cached
    
    user = client.users.get(user_id)
    cache.set(cache_key, user)
    return user
```

### Pagination

Always paginate large result sets:

```python
# List all users with automatic pagination
for user in client.users.list_all():
    process_user(user)
```
